##### Informações :  

ruby 2.6.2 | rails 5.2.8.1 | postgresql 14.5

App feito para estudos.

##### Execução :

$ bundle install | $ rails db:create | $ rails db:migrate | $ rails db:seed |
$ bundle exec rails s

##### Executar testes:

$ rspec  spec/models/user_spec.rb
$ rspec  spec/models/user_information_spec.rb
$ rails -c
